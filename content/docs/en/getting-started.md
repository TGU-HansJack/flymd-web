# Quick Start

Welcome to FlyMD! This guide will help you get started in minutes.

## Download and Install

Visit [GitHub Releases](https://github.com/flyhunterl/flymd/releases) to download the version for your operating system.

### Windows

Windows users can install via winget:

```bash
winget install flyhunterl.FlyMD
```

### macOS

Download the `.dmg` file and drag the app to Applications folder.

### Linux

Download the `.AppImage` file and make it executable.

## Basic Operations

- `Ctrl+N` - New file
- `Ctrl+O` - Open file
- `Ctrl+S` - Save file
- `Ctrl+E` - Toggle edit/preview
