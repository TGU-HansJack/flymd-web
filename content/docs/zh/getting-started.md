# 快速开始

欢迎使用飞速 Markdown！本指南将帮助您在几分钟内开始使用。

## 下载与安装

前往 [GitHub Releases](https://github.com/flyhunterl/flymd/releases) 下载适合您操作系统的版本。

### Windows

Windows 用户可以通过 winget 一键安装：

```bash
winget install flyhunterl.FlyMD
```

或者直接下载 `.exe` 安装包进行安装。

### macOS

下载 `.dmg` 文件，双击打开后将应用拖入 Applications 文件夹。

### Linux

下载 `.AppImage` 文件，添加执行权限后即可运行。

## 基本操作

- `Ctrl+N` - 新建文件
- `Ctrl+O` - 打开文件
- `Ctrl+S` - 保存文件
- `Ctrl+E` - 切换编辑/预览
